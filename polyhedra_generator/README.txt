#  GENERATION OF POLYHEDRA USING SAGE
#
# ==============================================================

1) Choose a KIND OF POLYHEDRA you want to generate

2) Launch a SageMath terminal session, you will get a prompt
   sage:

3) Execute the command
   sage: runfile demo_<KIND OF POLYHEDRA>.sage

   to see a 3D example of the kind of polyhedra

4) Optionally, edit and modify the file generate_<KIND OF POLYHEDRA>.sage

5) Execute the command
   sage: runfile generate_<KIND OF POLYHEDRA>.sage

6) The generated polyhedra are placed in
   polyhedra_generator/benchmark/<KIND OF POLYHEDRA>/
